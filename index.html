"Time of Day" is an interactive JavaScript project designed to adapt dynamically to the time of day. The website changes its background color and images to reflect whether it's morning, afternoon, evening, or night. It also displays a personalized greeting to match the time of day, making the experience more engaging. Additionally, the project includes a live display of the current day of the week and updates the year automatically to stay current.

This project showcases the use of JavaScript's Date object and DOM manipulation to create a dynamic, time-sensitive user experience.
